# eslint-plugin-no-autofix

## Install & usage

```bash
$ npm i eslint-plugin-no-autofix -D
```

add prefix "no-autofix" to the rulename in eslintrc:
```js
{
  "plugins": ["no-autofix"],
  "rules": {
    "no-autofix/no-debugger": "error"
  }
}
```
